Script simples para validar tipos de dados e propriedades de uma string em Python.



Dica: Sempre use () nos métodos (ex: .isalpha()) para executar a função, senão o Python mostrará apenas o local da memória.



*🇺🇸* English


Simple script to validate data types and string properties in Python.



Tip: Always use () with methods (e.g., .isalpha()) to execute the function, otherwise Python will only show the memory 
location.



*🇷🇴* Română


Script simplu pentru a valida tipurile de date și proprietățile unui string în Python.



Sfat: Folosiți întotdeauna () cu metodele (ex: .isalpha()) pentru a executa funcția, altfel Python va afișa doar locația memoriei.
